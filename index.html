Privacy Policy

RESIST AND DETOX does not collect, store, or share any personal user data.

The extension processes webpage content locally in the user's browser in order to replace predefined terms with alternative text.

To function, the extension sends the current website domain and browser language to a remote API in order to retrieve term replacement rules. This data does not contain any personal or identifiable information.

No tracking, analytics, or user profiling is performed.

All processing happens locally in the browser.
